Intro

Installable via npm install --save ipld-ethereum, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save ipld-ethereum

You can then require() ipld-ethereum as normal:

const ipldEthereum = require('ipld-ethereum')

In the Browser

Ipld-ethereum should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of ipld-ethereum, including index.js and index.min.js, are included in the /dist folder. Ipld-ethereum can also be found on unpkg.com under

  • https://unpkg.com/ipld-ethereum/dist/index.min.js
  • https://unpkg.com/ipld-ethereum/dist/index.js

resolve

util/createResolver.js

Resolves a path within a Ethereum block.

Returns the value or a link and the partial mising path. This way the IPLD Resolver can fetch the link and continue to resolve.

resolve
Parameters
binaryBlob (Buffer) Binary representation of a Ethereum block
path (string = '/') Path that should be resolved
Returns
Object: result - Result of the path it it was resolved successfully
Returns
any: result.value - Value the path resolves to
Returns
string: result.remainderPath - If the path resolves half-way to a link, then the remainderPath is the part after the link that can be used for further resolving
